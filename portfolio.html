Alternative Fuel Finder App Project

Usage:

A web App using multiple API's to generate data on local electric charging stations and alternative biofuel options based on a users current location. Upon entering your preferences and location, a modal will pop up displaying a map with markers of locations meeting that criteria, as well as a list of all locations and relavent information.

Technologies Used:

  • Bulma
  • NREL.gov API
  • Google Maps API
  • JQuery
